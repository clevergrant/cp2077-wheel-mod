## 🎛️ Essential Console Commands

[spoiler]
[size=3][b]Status & Information[/b][/size]
[code]
g923_status()              # Check mod status and hardware detection
g923_help()               # Show all available commands  
g923_config()             # Display current configuration
[/code]

[size=3][b]Configuration[/b][/size]
[code]
g923_sensitivity(1.5)     # Set steering sensitivity (0.1-5.0)
g923_deadzone(5, 10, 5)   # Set steering, throttle, brake deadzones
g923_curve("exponential") # Set steering curve (linear/exponential/s-curve)
g923_force_feedback(true) # Enable/disable force feedback
[/code]

[size=3][b]Vehicle-Specific Settings[/b][/size]
[code]
g923_vehicle_sensitivity(1.2, 0.8, 1.8)  # Car, motorcycle, truck sensitivity
[/code]

[size=3][b]Testing & Debug[/b][/size]
[code]
g923_debug(true)          # Enable debug mode with live input display
g923_test_effects()       # Test force feedback effects
g923_simulate(true)       # Enable simulation mode for testing
[/code]

[size=3][b]Configuration Management[/b][/size]
[code]
g923_save_config()        # Save current settings permanently
g923_reload_config()      # Reload configuration from file
g923_reset()              # Reset to default settings
[/code]
[/spoiler]

## 🛠️ Troubleshooting

[spoiler]
[size=3][b]❌ Wheel Not Detected[/b][/size]
[list=1]
[*]Check G HUB is running and wheel is recognized
[*]Use [font=Courier New]g923_status()[/font] to check connection
[*]Try [font=Courier New]g923_simulate(true)[/font] for testing without hardware
[*]Restart Cyberpunk 2077 and try again
[/list]

[size=3][b]❌ No Vehicle Response[/b][/size]
[list=1]
[*]Enable debug mode: [font=Courier New]g923_debug(true)[/font]
[*]Check for input values in console while driving
[*]Try manual override: [font=Courier New]g923_override(true)[/font]
[*]Verify CET is working properly
[/list]

[size=3][b]❌ Performance Issues[/b][/size]
[list=1]
[*]Disable debug output: [font=Courier New]g923_debug(false)[/font]
[*]Reduce sensitivity if too responsive: [font=Courier New]g923_sensitivity(0.8)[/font]
[*]Check performance monitoring: [font=Courier New]g923_performance("status")[/font]
[/list]

[size=3][b]❌ Force Feedback Not Working[/b][/size]
[list=1]
[*]Ensure G HUB force feedback is enabled
[*]Test effects: [font=Courier New]g923_test_effects()[/font]
[*]Check force feedback setting: [font=Courier New]g923_force_feedback(true)[/font]
[*]Verify G923 supports force feedback (it does!)
[/list]
[/spoiler]
